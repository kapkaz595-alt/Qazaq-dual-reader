# Qazaq Dual Reader

Qazaq Dual Reader 是一个支持**现代哈萨克文（西里尔字母）**与**传统哈萨克文
（阿拉伯字母）**双文字阅读的平台，覆盖电子书导入、自动文字转换、双文字同步
阅读、在线书城、浏览器插件与 PWA 等功能。

## 功能范围

- 📖 双文字阅读（西里尔 / 阿拉伯字母）
- 🔄 自动文字转换 + 一键切换整页文字
- 📚 电子书管理与 TXT / EPUB / PDF 导入
- 🔗 双文字同步阅读
- 🛒 在线书城
- 🧩 浏览器插件
- 📱 PWA（离线阅读支持）
- 🤖 AI 功能（OCR / TTS / 推荐 / 问答助手）

> 项目当前处于**工程启动阶段**，设计文档（PRD / 软件设计 / 目录结构 / 页面
> 设计 / 组件设计 / API 设计 / 数据库设计 / 状态管理设计 / 开发路线图）已全部
> 完成，详见 [`docs/`](./docs) 目录。

## 目录结构

项目采用前后端分离结构，`backend/modules/` 与设计文档中的 M1–M14 模块一一
对应，`frontend/modules/` 是对应模块的前端交互逻辑。

```
qazaq-dual-reader/
├── README.md
├── .gitignore
├── LICENSE
├── docs/              # 项目级文档（PRD、架构、软件设计等）
├── scripts/           # 跨前后端协同脚本（setup、ci-cd）
├── frontend/          # 前端项目（Web / PWA 共用基座）
└── backend/           # 后端项目（M1-M14 模块化）
```

完整目录树见 [`docs/directory-structure/`](./docs/directory-structure)。
